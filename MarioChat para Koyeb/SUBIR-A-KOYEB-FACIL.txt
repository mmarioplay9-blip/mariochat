PASOS FACILES PARA SUBIR MARIOCHAT

1. Abre GitHub:
   https://github.com/new

2. En Repository name escribe:
   mariochat

3. Marca Public.

4. Presiona Create repository.

5. En la pagina del repo, presiona:
   uploading an existing file

6. Arrastra TODO lo que esta en la carpeta:
   MarioChat para Koyeb

   Debes arrastrar:
   - public
   - server.js
   - package.json
   - Procfile
   - .gitignore
   - KOYEB.md

7. Presiona Commit changes.

8. Ve a Koyeb:
   https://app.koyeb.com

9. Create App > GitHub > elige el repo mariochat.

10. Configura:
    Run command: npm start
    Instance: Free

11. Presiona Deploy.

Cuando termine, copia el link .koyeb.app y mandaselo a tus amigos.
